# Signed Token API

A lightweight REST API built with Node.js and Express that generates and verifies time-bound, cryptographically signed, opaque reference tokens using JSON Web Tokens (JWT).

## Features

- Generate cryptographically signed tokens
- Verify token authenticity
- Time-bound token expiration
- Stateless authentication
- Input validation
- Secure secret key management using environment variables

## Tech Stack

- Node.js
- Express.js
- JSON Web Token (jsonwebtoken)
- dotenv

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
SECRET_KEY=yourSecretKey
PORT=3000
```

Run the server

```bash
node app.js
```

The server will start on

```
http://localhost:3000
```

## API Endpoints

### Generate Token

**POST**

```
/api/token
```

Request

```json
{
    "resourceId":"12345",
    "expiresIn":60
}
```

Response

```json
{
    "success":true,
    "token":"<JWT Token>"
}
```

---

### Verify Token

**POST**

```
/api/verify
```

Request

```json
{
    "token":"<JWT Token>"
}
```

Response

```json
{
    "success":true,
    "valid":true,
    "resourceId":"12345",
    "expiresAt":1752500000
}
```
